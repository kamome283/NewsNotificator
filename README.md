# 目的
.NET Aspire を活用し、複数のワーカーサービスを連携させる構成を試行・検証するためのニュースを通知するシステムです。

# 内容
本プロジェクトは、以下のコンポーネントで構成されます。

- **Aspire**: それぞれのコンポーネントを管理・連携させます。
- **RSS 取得サービス**: RSS フィードを取得し、データベースに保存します。
- **通知サービス**: データベース内のエントリーが指定条件を満たした場合、Discord などの外部サービスへ通知します。
- **Web フロントエンド**: ASP.NET Core を使用し、通知条件の設定などを行う管理画面を提供します。
- **コアコンポーネント**: 各サービスが共通で利用するドメインロジックや基盤処理を定義します。
